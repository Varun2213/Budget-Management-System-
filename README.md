# Budget Management System

A web-based Budget Management and MIS Reporting System developed using Google Apps Script, HTML, CSS, JavaScript, and Google Sheets.

## Features

- Budget Entry Management
- Actual Expense Tracking
- Variance Analysis
- KPI Dashboard
- Multi Financial Year Comparison
- Consolidated Reporting
- Approval Workflow System
- User & Role Management
- Audit Log Tracking
- Dashboard Charts & Analytics
- Import / Export Support
- Google Sheets Integration

## Technologies Used

- Google Apps Script
- JavaScript
- HTML5
- CSS3
- Chart.js
- Google Sheets

## Key Functionalities

- Automated MIS Reporting
- Dynamic Dashboard Generation
- Department-wise Budget Tracking
- Approval & Revision Workflow
- Financial Data Analysis
- User Access Control
- Real-time Reporting
